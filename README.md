# TAN LAB Jekyll site

This package is preconfigured for the repository:

`https://hanzeyan.github.io/drummondlab.github.io/`

## Current config

- `url: https://hanzeyan.github.io`
- `baseurl: /drummondlab.github.io`

## How to use

1. Delete the old files in your GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set **Source** to **GitHub Actions**.
5. Wait for the workflow to finish.

## If you later move to a custom domain or `username.github.io`

Update `_config.yml`:

```yml
url: "https://your-domain-or-username.github.io"
baseurl: ""
```
